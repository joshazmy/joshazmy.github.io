Claude Usage Gauge

hardware · bring-up

A desk gadget that shows my live Claude usage as pixel art on a 2" screen — and flashes when a session needs me: amber when Claude Code is waiting on me (a permission prompt or idle input), green when a long turn finishes while I’m away. A pixel creature’s face escalates calm → focused → worried → dead-eyed above two color-by-fill usage bars. Built the full stack: Claude Code hooks → a beacon state file, a stdlib usage reader, a documented design system, a 1:1 browser simulator of the panel, and ESP32-S3 firmware (Arduino_GFX, PSRAM framebuffer, a serial protocol with an explicit stale-data state). Hardened through multi-agent and adversarial Codex reviews before shipping.

ESP32-S3 · C++ / Arduino_GFX · Python · HTML Canvas · Claude Code hooks
